# React 版本 - 傅冠豪醫師個人首頁

使用 React 18 + Vite 建構的個人首頁。

## 開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 構建生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 部署

當前配置為自動部署到 GitHub Pages 根目錄 (https://haobbc.github.io/)

推送到 main 分支後，GitHub Actions 會自動構建並部署。

更多部署選項請參考根目錄的 [README.md](../README.md)。

## 技術棧

- React 18
- Vite 7
- CSS (原生樣式)
